## 6. L1 本地路网（M8-R3 · 维护说明）

**OSM 真实建筑/道路（推荐）**

从 `data/map.osm` 导出 87 个建筑面 + 30 条真实道路，生成 `data/pilot_osm_geo.json`：

```bash
python scripts/carla/osm_to_pilot_geo.py
```

后端 `OsmPilotGeoRepository` 加载该文件后：
- **禁行校验**使用 OSM 真实建筑面（不再用简化 6 格矩形）
- **路径规划**在 OSM 道路 + 不穿建筑的中排走廊上网格最短路
- **禁止直线兜底**：无法规划合法路线时 `routeInvalid=true`，前端不画穿楼线

无高德 Key 时依赖上述本地路网；有 Key 时优先高德驾车，若穿建筑则回退本地路网。
