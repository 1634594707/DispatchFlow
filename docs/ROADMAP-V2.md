### 家纺垂直 — 找家纺 × 新石器

| 维度 | 差距 | Phase |
|------|------|-------|

### Phase 14 — 家纺 / 产业带垂直增强 · 可选 · ~3 周

> **M4**：线路 + 枢纽 + 旺季 demo

#### 14.1 线路化调度
- [ ] 数据模型：线路 = 有序站点 + 运营时段 + 园区
- [ ] 线路 CRUD、启用/停用
- [ ] 订单绑定线路模板（移动下单 + 管理端）
- [ ] 工作台按线路筛选任务池
- [ ] 线路 occupancy / 简化动态发车

#### 14.2 枢纽 / 母港 / 缓冲
- [ ] 站点类型 HUB / BUFFER / MOTHERSHIP + 容量上限
- [ ] 派车评分枢纽容量惩罚 → `HUB_CAPACITY_FULL`
- [ ] 工作台母港分流视图（排队任务、改派建议）
- [ ] Webhook：`HUB_ARRIVAL` / `HUB_DEPARTURE`（AGV 协同预留）

#### 14.3 能量策略（充电 + 换电）
- [ ] 换电柜 / 换电会话模型
- [ ] 车辆状态：充电中 / 换电中 / 插枪待命
- [ ] 低电 → 充电 vs 换电策略配置
- [ ] 充电报表区分充电/换电时长

#### 14.4 自动化规则引擎（非大模型）
- [ ] IF-THEN 规则 UI（如 SOC&lt;20% → 回充任务）
- [ ] 规则变更审计
- [ ] 规则触发高峰模式（联动 11.4）

#### 14.5 高峰 / 旺季预案
- [ ] 高峰模式开关（手动 + 定时）
- [ ] 预案模板：「家纺大促」「日常」
- [ ] 高峰 vs 平日对比报表
- [ ] 监控大屏「生产 / 高峰」模式切换

#### 14.6 多车型混编预留
- [ ] 车辆类型 OUTDOOR_L4 / INDOOR_AMR / FORKLIFT
- [ ] 派车按类型过滤
- [ ] `FleetAdapter` 注册表
- [ ] VDA5050 适配评估文档（Phase 15 候选）

#### 14.7 远程管控 + 现场运维

- [ ] 异常指派现场运维工单
